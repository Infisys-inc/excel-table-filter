## Excel Table Filter Plugin

A Vanilla JS plugin to generate a checkbox drop-down table-filter.

![Excel Table Filter Example](https://raw.githubusercontent.com/Infisys-inc/excel-table-filter/refs/heads/main/excel-table-filter.png)
*Excel Table Filter Example*

## Usage

- Install the dependencies by running: `npm install`.
- Build the project by running: `npm run build`.
- Add `dist/excel-table-filter-bundle.min.js` and `dist/excel-table-filter-style.css` to your project
- Render the dropdown to a`#table`: <br>`new ExcelTableFilter(document.getElementById('table'))`
